php5.4 织梦dedecms后台登录空白的解决方法

linux，php5.4， 织梦dedecms后台登录空白，原因：

include/userlogin.class.php，里面有一个keepuser()函数，是用session_register来注册一个session变量，但是在php5.4中已经移除了这个功能

把@session_register($this->keepUserIDTag);    注释掉，然后改为

if (!isset($_SESSION[$this->keepUserIDTag]))

如下：

if (!isset($_SESSION[$this->keepUserIDTag]))

//@session_register($this-&gt;keepUserIDTag);

 

附，nginx日志报错：

nginx fastcgi &ldquo;Primary script unknown&rdquo; when configuring userdir

修改：

&nbsp;&nbsp;&nbsp; location ~ \.php$ {

&nbsp;&nbsp;&nbsp; ...

&nbsp;&nbsp;&nbsp; fastcgi_param SCRIPT_FILENAME $document_root$fastcgi_script_name;

&nbsp;&nbsp;&nbsp; ...

&nbsp;&nbsp;&nbsp; }

为：

&nbsp;&nbsp;&nbsp; location ~ \.php$ {

&nbsp;&nbsp;&nbsp; ...

&nbsp;&nbsp;&nbsp; fastcgi_param SCRIPT_FILENAME $request_filename;

&nbsp;&nbsp;&nbsp; fastcgi_param REQUEST_URI $uri?$args;

&nbsp;&nbsp;&nbsp; ...

&nbsp;&nbsp;&nbsp; }
