web服务器使用mod_deflate（apache）gzip（nginx）压缩节约带宽及测试方法

  cpu速度越来越快也更便宜，IDC机房带宽很昂贵，所以用cpu来换取带宽。

apche启用mod_deflate压缩：

&lt;ifmodule mod_deflate.c&gt;

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; DeflateCompressionLevel 9

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; SetOutputFilter DEFLATE

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; SetEnvIfNoCase Request_URI \.(?:gif|jpe?g|png)$ no-gzip dont-vary

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; SetEnvIfNoCase Request_URI \.(?:exe|t?gz|zip|bz2|sit|rar)$ no-gzip dont-vary

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; SetEnvIfNoCase Request_URI \.pdf$ no-gzip dont-vary

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; BrowserMatch ^Mozilla/4 gzip-only-text/html

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; BrowserMatch ^Mozilla/4\.0[678] no-gzip

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; BrowserMatch \bMSIE !no-gzip !gzip-only-text/html

&lt;/ifmodule&gt;

nginx 启用gzip压缩：

http {

&nbsp;&nbsp; gzip&nbsp; on;

&nbsp;&nbsp; gzip_min_length&nbsp; 1000;

&nbsp;&nbsp; gzip_buffers&nbsp;&nbsp;&nbsp;&nbsp; 4 8k;

&nbsp;&nbsp; gzip_types&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; text/plain application/x-javascript text/css text/html application/xml;

&nbsp;&nbsp; }

压缩效果测试

不使用压缩:

curl http://veryi.com/w/1.html --silent --write-out &quot;size_download=%{size_download}\n&quot; --output /dev/null

size_download=64031

使用 Accept-Encoding 头：

curl http://veryi.com/w/1.html --silent -H &quot;Accept-Encoding: gzip,deflate&quot; --write-out &quot;size_download=%{size_download}\n&quot; --output /dev/null

size_download=20012

比较数字就可以知道压缩效果。

使用 HTTP 1.0 测试：

curl http://veryi.com/w/1.html --silent --http1.0 -H &quot;Accept-Encoding: gzip,deflate&quot; --write-out &quot;size_download=%{size_download}\n&quot; --output /dev/null

size_download=64031
