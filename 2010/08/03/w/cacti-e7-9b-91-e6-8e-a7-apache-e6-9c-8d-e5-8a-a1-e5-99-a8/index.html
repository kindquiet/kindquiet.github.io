cacti 监控 apache 服务器

模板下载：

[http://forums.cacti.net/about9861.html](http://cacti 监控 apache 服务器)

安装脚本

wget http://forums.cacti.net/download.php?id=9907

gzip -d ws_apachestats.pl.gz

cp apache.apache ws_apachestats.pl &lt;path_cacti&gt;/scripts/

chmod 0755 &lt;path_cacti&gt;/scripts/apache.apache ws_apachestats.pl 

下载 xml 文件

http://forums.cacti.net/download.php?id=12524

在cacti控制台把 cacti087b_host_template_webserver__apache_444.xml 导入 cacti

客户机配置

&lt;IfModule mod_status.c&gt;

&lt;location /server-status&gt;

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; SetHandler server-status

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; Order Deny,Allow

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; Deny from all

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; Allow from 127.0.0.1

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; Allow from 192.168.8.8

&lt;/location&gt;

监控服务器(192.168.8.8)测试：

./ws_apachestats.pl clint_ip
